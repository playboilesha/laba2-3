Тест кейс №1
Задача: Добавить в корзину  черную майку мужскую на сайте "https://aliexpress.ru/"
1. Зайти на сайт "https://aliexpress.ru/"
2. Ввести в строку поиска "мужская черная майка"
3. Нажать клавишу "Enter"
4. Перейти на понравившийся товар
5. Выбрать черный цвет
4. Нажать на кнопку "добавить в корзину"
Ожидаемый результат: Вы добавили товар в корзину

Тест кейс №2
Задача: найти чехол на телефон 11 айфон по самой дешевой цене
1. Зайти на сайт "https://aliexpress.ru/"
2. Ввести в строку поиска "чехол айфон 11"
3. Нажать клавишу "Enter"
4. В строке "Сортировка по:" нажать на вкладку "Цене"
Ожидаемый результат: Вы увидели чехол на 11 айфон по самой дешевой цене
